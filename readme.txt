== Description ==

This plugin is very simple to use, provide a shortcode to create player. You can create player in any where of your site.

== Installation ==

1. Upload plugin folder to the `/wp-content/plugins/` directory
2. Activate the plugin through the 'Plugins' menu in WordPress
3. Use shortcode in your posts/pages

* Base structure of shortcode:
`[jwp video="" thumb="" title="" subtitle=""]`

* Full option shortcode:
`[jwp video="" thumb="" title="" subtitle="" logo="" posisi=""]`

Example:
`[jwp video="http://distribution.bbb3d.renderfarming.net/video/mp4/bbb_sunflower_1080p_30fps_normal.mp4" thumb="https://peach.blender.org/wp-content/uploads/bbb-splash.png" title="" subtitle="https://raw.githubusercontent.com/Dador/JavascriptSubtitlesOctopus/gh-pages/subtitles/test.ass" logo="https://static.wikia.nocookie.net/logopedia/images/e/ea/Big_Buck_Bunny.svg" posisi="bottom-right"]`
